# Bibliography

This directory stores papers and references used by the ORACLE/MATRIX design.

- `morbvis-browser-based-molecular-orbital-visualization-with-webgpu-accelerated-on-the-fly-evaluation.pdf`: MOrbVis reference paper for browser-based molecular-orbital visualization. The GUI integration launches the public application at `https://yasuaki-ito.github.io/morbvis/`; source code is available at `https://github.com/Yasuaki-Ito/morbvis`.
