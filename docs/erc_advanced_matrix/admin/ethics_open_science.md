# Ethics, Open Science And Data

MATRIX is primarily computational, but the ethics and open-science sections
still need a deliberate plan.

## Likely Ethics Topics

- No human participants unless later demonstrators introduce external data.
- No animal work expected.
- No personal data expected.
- Dual-use risk appears low but should be checked for any materials/design
  applications.
- Software, data licensing and third-party database terms must be respected.

## Open Science Plan

- Release benchmark corpora where licensing allows.
- Publish schemas and provenance records.
- Archive software releases with persistent identifiers.
- Publish spectra/tables with metadata sufficient for reproduction.
- Use open access publication routes.

## Research Data Management

Data classes:

- molecular geometries and enriched `xyzin` states;
- quantum-chemistry outputs where redistribution is allowed;
- generated spectra and thermochemical tables;
- benchmark reference results;
- scripts and manifests needed for reproducibility.

Open questions:

- Which QM output files can be redistributed?
- How to store large generated corpora?
- Which external experimental databases allow cached redistribution?
