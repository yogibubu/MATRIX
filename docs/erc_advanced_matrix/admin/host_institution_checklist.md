# Host Institution Checklist

To confirm with the host institution research office.

## Immediate

- Internal deadline for ERC-2026-AdG.
- Required internal approval workflow.
- Host Institution support letter process.
- Budget office contact for lump-sum planning.
- Personnel cost tables.
- Ethics and data-management support.
- Open access and research-data-management policies.

## Proposal Forms

- Administrative forms.
- Host Institution commitment.
- Ethics Issues Table and self-assessment.
- Research-data-management/open-science information.
- Budget/lump-sum tables.
- Any local declarations required by the institution.

## Submission

- Funding & Tenders Portal access.
- PIC and host details.
- Authorized representative workflow.
- Final upload deadline earlier than official deadline.
