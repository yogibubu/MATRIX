# Budget And Lump-Sum Planning

ERC-2026-AdG is implemented as a lump-sum pilot according to the official ERC
Advanced Grant page. This file is a planning worksheet, not the official
budget.

## Budget Principles

- Tie every cost to a scientific work package.
- Justify personnel through methodological deliverables and demonstrators.
- Include computational resources only when they are essential for frontier
  objectives.
- Include open access, data management, dissemination and software maintenance
  where eligible and necessary.
- Keep any major equipment or large-facility request explicitly connected to
  the scientific breakthrough.

## Non-Personnel Cost Buckets

- High-performance computing.
- Storage for benchmark corpora and generated spectra.
- Open access and publication costs.
- Travel for collaborations, conferences and training.
- Workshops or user/developer meetings.
- Optional equipment or large-facility costs if scientifically justified.
