# Scientific Innovations To Emphasize

MATRIX must not be presented as only a technological or implementation
advance. The proposal should explicitly separate infrastructure from original
scientific methodology.

## New Methodological Components

- Localized generalized internal coordinates that retain chemical
  interpretability in large, flexible and weakly bound systems.
- Symmetry-adapted generalized internal coordinates built from group
  operations without mixing incompatible coordinate types.
- Special coordinates for fragments, virtual centres, rings, bond centres and
  relative orientations, with analytic B-matrix construction.
- Chemically controlled primitive-coordinate selection that removes redundancy
  while preserving chemically essential or symmetry-essential coordinates.
- Electronic-structure PCS/composite strategies generalized to a broader
  contribution-based accuracy hierarchy.
- Persistent-state VPT2/VCI workflows in which anharmonic treatments inherit
  geometry, topology, symmetry, normal-mode identity, force fields and
  provenance.
- Local-mode and fragment-aware anharmonic models for molecules beyond the
  current high-accuracy size frontier.
- Knowledge-based error decomposition before DeltaML residual correction.
- Intra-/inter-fragment level partitioning for energies, structures,
  vibrational corrections and force fields.

## ERC Framing

The implementation is necessary because these methods need a shared molecular
state, robust parsers, reproducible provenance and hardware-aware execution.
But the scientific claim is stronger: MATRIX introduces new coordinate,
symmetry, anharmonic, fragment-partitioning and error-decomposition methodology
for predictive spectroscopy of light-element molecules up to about 100 atoms.

## Text To Avoid

Avoid describing MATRIX as a "software suite", "workflow manager" or "GUI
around existing codes" without immediately stating the original scientific
methods. Those descriptions undersell the ERC novelty.
