# PI Track-Record Strategy

## Main Message

MATRIX is broad, but the PI's CV should make that breadth credible. The track
record must show the ability to coordinate theory, algorithms, implementation,
validation, applications and experiment without losing scientific focus.

The retirement point should be handled proactively. The argument is not that
the PI remains active despite retirement; it is that retirement has increased
scientific focus by removing administrative distractions. Recent productivity
and recent concepts must show that MATRIX is driven by current scientific
creativity, not only by past reputation.

## Narrative Pattern

Use a repeated pattern across selected achievements:

1. A theoretical or methodological problem was identified.
2. A new model, approximation or algorithm was developed.
3. The method was implemented robustly enough to be reused.
4. The result was tested against experimental observables.
5. The theory-experiment comparison improved either the interpretation or the
   method itself.

This pattern directly supports the MATRIX claim that theory and experiment are
not separate endpoints but part of one feedback loop.

## Evidence To Collect

- Selected publications in theoretical and computational spectroscopy.
- Work on semiexperimental and empirical-equilibrium structures.
- Studies where rotational, vibrational or electronic spectra were compared
  directly with experiment.
- Method-development papers that produced reusable codes or protocols.
- Collaborative papers with experimental groups.
- Previous grants, networks or projects requiring coordination of multiple
  scientific components.
- Supervision record across theory, coding and applications.

## B1 Usage

The B1 CV/track-record section should not list achievements passively. For
each selected item, add a short MATRIX relevance sentence:

- what scientific layer it de-risks;
- which MATRIX work package it supports;
- how it demonstrates control of theory-experiment feedback.

## Panel Message

The intended panel reaction should be:

> This is a broad project, but the PI has already demonstrated the ability to
> connect high-level theory, practical computational tools and experimental
> validation across molecular spectroscopy.
